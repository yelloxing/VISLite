let colors = getLoopColors(num, alpha=1)

formatColor

0.4.1 新增，且只支持Web端，如果后续有需要会进行兼容扩展

引入 

import { formatColor } from "vislite"

语法

对颜色进行格式化处理： 

let colorArray = formatColor(color, format)

color表示颜色值，应该是一个合法的css颜色字符串；format表示目标格式，也是一个字符串，可以有如下选择：

  1. "3d"：表示用于3d绘图，返回的就是一个数组，例如：[1, 0, 0, 1]，表示红色；
  2. "rgba"：返回颜色的rgba格式，例如：rgba(255, 0, 0, 1)，表示红色；
  3. "rgb"：返回颜色的rgb格式，例如：rgb(255, 0, 0)，表示红色，如果传递的颜色有透明度，会被省略；
  4. "hexAlpha"：带透明度的十六进制颜色，例如：#ff0000ff表示红色；
  5. "hex"：十六进制颜色，例如：#ff0000，表示红色，如果传递的颜色有透明度，会被省略。
