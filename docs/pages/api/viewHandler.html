控制信息捕获
0.2.0 新增，且只支持Web端，如果后续有需要会进行兼容扩展

引入 

import { viewHandler } from "vislite"

语法 

viewHandler(function(params){
    // todo
})

当鼠标或者键盘进行操作的时候，会触发回调，回调函数有一个形参，格式如下： 

params: {
    type: string
    value: number
    normal: number[],

    // v0.4.1开始支持
    event: Event   
}

其中type表示操作类型，具体可能值如下：

值 类型 说明
lookUp 向上 按下键盘向上方向键触发
lookDown 向下 按下键盘向下方向键触发
lookLeft 向左 按下键盘向左方向键触发
lookRight 向右 按下键盘向右方向键触发
rotate 旋转 使用鼠标左键按下拖动的时候触发，此时normal表示滚动法向量，满足右手螺旋法则，value表示旋转弧度
hover 悬浮 在鼠标左键没有按下的情况下移动鼠标的时候触发（v0.4.1开始支持）
scale 缩放 滚动鼠标滚轮的时候触发，此时value表示缩放倍数
