纹理
0.3.0 新增

引入 

    import { Texture } from 'vislite'

语法

获取纹理实例： 

    let texture = new Texture(painter, type, unit)

其中参数解释如下：

  1. painter：WebGLRenderingContext类型
  2. type：string类型，可以是"2d"或"cube"，分别表示是二维纹理还是立方纹理
  3. unit：number类型，表示纹理单元，默认0

方法

useImage

链接图片资源(单一)，表示纹理图片，TexImageSource类型： 

    texture.useImage(image)

useCube

链接图片资源（多张）： 

    texture.useCube(images, width, height)

其中参数解释如下：

  1. images：纹理图片集，是一个数组，里面的元素是TexImageSource类型
  2. width：图片宽
  3. height：图片高
