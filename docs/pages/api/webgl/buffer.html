缓冲区
0.3.0 新增

引入 

    import { Buffer } from 'vislite'

语法

获取缓冲区实例： 

    let buffer = new Buffer(painter, isElement)

其中参数解释如下：

  1. painter：WebGLRenderingContext类型
  2. isElement：boolean类型，默认false，表示不是索引缓冲区

方法

use

使用这个缓冲区对象： 

    buffer.use()
温馨提示：应该先于写入write和分配divide。

write

写入数据，数据data类型可以是 Uint8Array | Int16Array | Uint16Array | Int32Array | Uint32Array | Float32Array： 

    buffer.write(data)

divide

分配使用 

    buffer.divide(location, size, stride, offset)

其中参数解释如下：

  1. location：number类型，表示变量位置
  2. size：number类型，表示一个数据的长度
  3. stride：number类型，表示一组数据的长度(可能包含多组)
  4. offset：number类型，表示读取位置偏移，可选，默认0
