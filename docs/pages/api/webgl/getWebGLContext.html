画笔
0.3.0 新增，且只支持Web端，如果后续有需要会进行兼容扩展

引入 

    import { getWebGLContext } from 'vislite'

语法

获取画笔实例，其中el就是挂载点，比如一个div： 

    let painter = getWebGLContext(el, scale, mode)

其中参数解释如下：

  1. el：页面挂载点，比如DIV节点
  2. scale：画布缩放倍数，可选，默认1
  3. mode：设置画布缩放模式，可选（v0.8.0开始支持）

返回的画笔painter类型是WebGLRenderingContext，此画笔上具体方法你可以 。
