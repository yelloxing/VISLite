着色器
0.3.0 新增

引入 

    import { Shader } from 'vislite'

语法

传递一个WebGLRenderingContext类型的参数painter即可获取着色器实例： 

let shader = new Shader(painter)

属性

program

着色器程序。

方法

use

使得当前的着色器程序启用： 

    shader.use()
温馨提示：需要在编译compile完成后再调用这个方法。

compile

编译具体的着色器程序，需要传递的是两个参数，分别是顶点着色器和片段（元）着色器的字符串代码： 

    shader.compile(vshaderSource, fshaderSource)

内置程序

此外，你还可以使用内置定义好的着色器程序，语法如下： 

    shader.compile(shaderName)
0.4.1 新增

那具体有哪些内置的着色器，下面一一说明：

color

可以绘制单一颜色物体：

  • u_matrix：uniform mat4类型，用于设置物体变换矩阵；
  • u_color：uniform vec4类型，用于设置物体颜色；
  • a_position：attribute vec4类型，用于设置物体点的位置。
colors

可以绘制每个点独立设置颜色的物体：

  • u_matrix：uniform mat4类型，用于设置物体变换矩阵；
  • a_color：attribute vec4类型，用于设置物体颜色；
  • a_position：attribute vec4类型，用于设置物体点的位置。
image

可以使用纹理图片贴在物体表面：

  • u_matrix：uniform mat4类型，用于设置物体变换矩阵；
  • a_textcoord：attribute vec2类型，用于设置UV纹理坐标；
  • u_sampler：uniform sampler2D类型，用于设置纹理单元；
  • a_position：attribute vec4类型，用于设置物体点的位置。
cube

可以使用立法纹理贴在物体表面：

  • u_matrix：uniform mat4类型，用于设置物体变换矩阵；
  • a_normal：attribute vec4类型，用于设置物体的法向量；
  • u_texture：uniform samplerCube类型，用于设置纹理单元；
  • a_position：attribute vec4类型，用于设置物体点的位置。
